# UrLink Chrome Extension

# Get start

`development` 환경에서는 `chrome api` 기능을 사용 할 수 없어 `build`를 하고 크롬 익스텐션 개발 모드로 확인 가능합니다. 

## development 
```
npm i
npm run start
```

## production [build]
```
npm i
npm run build
```
## chrome development
```
 1. 크롬 실행
 2. 확장 프로그램
 3. 압축해지된 확장 프로그램을 로드합니다[클릭]
 4. build 된 폴더 선택
 ```
